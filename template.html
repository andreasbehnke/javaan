Javaan

Java Static Code Analyser

  • Overview
  • Documentation
  • Release Notes
Download this project as a .zip file Download this project as a tar.gz file

Javaan is a command line tool for analysing java bytecode. It is not bound to the Java programming language but supports parsing of byte code from any java virtual machine based programming language. Javaan helps fixing broken builds which produce archives with duplicate classes. It provides an overview about call graphs, class dependencies and class hierarchy. Javaan helps analysing weaknesses of software design by providing information about dependency cycles and library entry methods.

Javaan uses the apache BCEL library for parsing java byte code and the graph library JGraphT for building the dependency graph and class graph.

Installation

  • Make sure you have latest version of java installed
  • Download binary archive as tar.gz or zip file.
  • Extract archive contents
  • Add javaan.sh to your systems path variable.
  • Run javaan to see a list of all available commands
