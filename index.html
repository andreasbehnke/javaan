Javaan

Java Static Code Analyser

Download this project as a .zip file Download this project as a tar.gz file

Javaan is a command line tool for analysing java bytecode. It is not bound to the Java programming language but supports parsing of byte code from any java virtual machine based programming language. Javaan helps fixing broken builds which produce archives with duplicate classes. It provides an overview about call graphs, class dependency and class hierarchy.

Javaan uses the apache BCEL library for parsing java byte code and the graph library JGraphT for building the dependency graph and class graph.
